## 2024-05-23 - Settings Panel Accessibility & Web Parity
**Learning:** Extracting complex UI sections (like settings panels) into dedicated components with `aria-label`, `aria-controls`, and `aria-expanded` attributes significantly improves accessibility structure and maintainability. When features are platform-specific (e.g., native-only image adjustments), explicitly disabling controls with a clear warning is a critical UX pattern to prevent user confusion, rather than allowing silent failures.
**Action:** Always verify platform capabilities before enabling UI controls and ensure extracted components carry full accessibility metadata.
